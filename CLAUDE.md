This is the GitHub profile README repo for Farhan Rafly Fahrezi S (fahrezis).

## Updating the README

### Writing

The "Writing" section in README.md lists the 3 most recent blog posts from [fahrezis.com](https://fahrezis.com).

To update:
1. Fetch fahrezis.com to get the latest blog posts
2. Replace the 3 entries under "Recent posts:" with the new titles and URLs
3. Keep the format: `- [Post Title](https://fahrezis.com/path-to-post)`

### Talks & Community

The "Talks & Community" section lists speaking engagements and community contributions, ordered most recent first.

To update:
1. Add new entries at the top of the list
2. Use the format: `- **Talk Title** — Event Name, Mon YYYY`
3. Non-talk contributions (mentoring, organizing) go at the bottom of the list

Source for talks and community activities:
- fahrezis.com (talks/speaking section)
